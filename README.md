# Pong - The Game
Pong is written in plain JavaScript.

The one who first reaches 5 goals wins.

## Controls

* Player 1
    * W = Move Up
    * S = Move Down
* Player 2
    * Arrow Up = Move Up
    * Arrow Down = Move Down
